Introduction

Codotronix Game Library (or CXGL in short) is a very minimalistic game development Library for browsers. It is a Javascript based Library and it limits its scope in Javascript only. Hence it can be used with DOM / SVG / Canvas or anything. It is a very small library and currently the minified JS file is only 4 KB in size. It provides basic functionalities like,

  • creating / grouping / querying game objects
  • binding functions to keyboard keys
  • a game ticker for repititive events
  • collision detection

Get It

Get the js and minified js file from Github, inside dist folder
