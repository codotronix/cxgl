cxgf.GameObject

APIs

cxgf.GameObject.create(gameObjectConfig) Creates and returns a new Game object. To create the new game object it takes a JSON parameter called gameObjectConfig.
  1. x: <Number> X Coordinate
  2. y: <Number> Y Coordinate
  3. speed: <Number> Defaults to 1
  4. speedX: <Number> Defaults to speed
  5. speedY: <Number> Defaults to speed
  6. htmlId: ID of HTML DOM element that represents this object
  7. gravityOn: true / false to turn gravity ON / OFF on this object
  8. holdDirectionNTurns: Used for random moves only. It indicates how many turns or ticks to wait before randomizing the direction again. Defaults to 10
  9. onCollision (collidingGameObject): <Function> if this gameObject is added to cxgf.Collision.watch list, then whenever a collision is detected, this onCollision function will be called with the collidingGameObject as its parameter
  10. type: <string> A string specifying the meaning / type / category of this object. For example, if it is a game of "cats" and "mice", we can give all the cat objects a 
    type: cat
    and each mouse objects a 
    type: mouse
    , so that, later we can easily get all the same type of objects by one simple query like, 
        cxgf.GameObject.getAll({
        type: "car"
    })

    see cxgf.GameObject.getAll(identifier: JSON) next for more details
cxgf.GameObject.getAll(identifier: JSON) Returns all the Game Objects that match(es) the provided identifier
  1. cxgf.GameObject.getAll() - when identifier is blank, it will return all the GameObjects
  2. "identifier" can be a json 
        cxgf.GameObject.getAll({
        type: "car",
        color: "red"
    })
    It will return all the red car GameObjects
<GameObjectInstance>.move(direction) Move the object in the given direction. "direction" can be any one of the following strings
  1. "W" or "LEFT"
  2. "E" or "RIGHT"
  3. "N" or "UP"
  4. "S" or "DOWN"
  5. "RANDOM-4" (Default): Randomly move to any direction among E / W / N / S
  6. "RANDOM-8": Randomly move to any direction among E / W / N / S / NW / NE / SW/ SE (PENDING)
  7. "RANDOM-EW": Randomly move to any direction among E / W (PENDING)
  8. "RANDOM-NS": Randomly move to any direction among N / S (PENDING)
