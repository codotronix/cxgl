cxgf.Collision

APIs

cxgf.Collision.watch (gameObject1, gameObject2)
OR
cxgf.Collision.watch (gameObjectArray1, gameObjectArray2) 		Add certain game gaobjects to the collision watch. Whenever a collision is detected on any of those object, the "onCollision" function (if defined) will be called on that object. See respective documentation on cxgf.GameObject.create(gameObjectConfig) for more details on onCollision function
cxgf.Collision.isColliding (gameObject1, gameObject2)
OR
cxgf.Collision.isColliding (gameObjectArray1, gameObjectArray2)
returns boolean 		returns true / false depending on whether there currently is a collision
