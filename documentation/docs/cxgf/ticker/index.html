cxgf.Ticker

APIs

cxgf.Ticker.onTick (callbackFn, bindingObj, skipInterval)
returns tickListenerID 		A function to be executed on each tick of the game cycle, probably lies somewhere between 30fps to 60fps
  1. callbackFn: The function to be executed each time a tick happens
  2. bindingObj: The object on which the abve mentioned callbackFn to be executed, to preserve the original "this" object
  3. skipInterval: (number) If our function needs not be called in each tick, but to be skipped certain number of ticks, we will pass that Integer (whole number) here

It returns a tickListenerID which can be used later to remove a callback function from being executed on ticks
cxgf.Ticker.removeTick (tickListenerID) If we want a function (say FnX) to be removed from tick callback list, we can do that by using this function. We just need to pass the tickListenerID which we got back from cxgf.Ticker.onTick while adding FnX to the tick listener list.
cxgf.Ticker.startTick () Use this function to start the ticker
cxgf.Ticker.stopTick () Use this function to stop the ticker
