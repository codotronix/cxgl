cxgf.KeyEvent

APIs

cxgf.KeyEvent.onKeyPress (cxgf.KeyEvent.keys.KEY_NAME, callbackFn, bindingObject) This function binds a user defined callback function to a keyboard keypress event. We can also pass the binding object as 3rd parameter if we want to use the original 'this' variable inside that function when the callback happens.
Use case can be, say, we have a "moveLeft" function which should be executed whenever the player presses the "LEFT" arrow key on the keyboard, so we would define it like this, 
cxgf.KeyEvent.onKeyPress(cxgf.KeyEvent.keys.LEFT, player.moveLeft, player);

  1. cxgf.KeyEvent.keys.KEY_NAME: a predefined value for each keyboard keypress. Details are below in its own section
  2. callbackFn: A callback function that will be executed each time the keyboard key is pressed (e.g. player.moveLeft in our above example)
  3. bindingObject: If the above callback function is defined on sone object and we want to use some other properties defined on that object (say, speed or spaceshipColor etc), we need to pass the original object to retain the actual value of "this" when the callback happens (e.g. "player" in our above example).
cxgf.KeyEvent.keys.KEY_NAME List of keyboard keys available for binding
